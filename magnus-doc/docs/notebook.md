## Accéder aux notebooks

Les notebooks contiennent l'essentiel des données d'analyse du dataset utilisé pour entrainer l'IA

Ces notebooks sont accesibles depuis la racine du projet dans le dossier notebooks

### eda_cleaning Notebook

Ce notebook contient l'EDA et le cleaning de la dataset.

### result_analysis Notebook

Ce notebook contient l'analyse de nos différents modèles obtenus après tuning des hyperparamètres. A noté qu'avant de run ce notebook, il est nécessaire d'extraire les modèles contenus dans results_datanotebook.rar

### optuning Notebook

Ce notebook est un notebook de test pour la pipeline de nettoyage de données et le training du modèle avec XGBoost
