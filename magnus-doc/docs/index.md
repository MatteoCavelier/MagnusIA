# Bienvenue sur la documentation

Bienvenue sur la documentation de Magnus IA.

Naviguez dans les sections pour installer le projet, démarrer rapidement ou apprendre à optimiser les modèles.

## A propos

Magnus IA est une IA permettant de prédire des résultats de matchs d'échecs à partir des données suivantes :

* Elo du joueur Blanc
* Elo du joueur Noir
* Format de temps du match (en code d'incrémentation : 10+5 par exemple)
* Nombre de tours estimé pour atteindre l'ouverture choisie
* Le code ECO de l'ouverture

Une fois les données envoyées, le modèle va automatiquement prédire le joueur le plus enclin à remporter la partie et le renvoie
